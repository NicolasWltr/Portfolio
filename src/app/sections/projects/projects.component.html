A Variation of the classic game Tic-Tac-Toe, with more complex game modes, that allows you to play with anyone around the world or with 2 players on one device.

Simply click here or the LIVE PROJECT button below to open the game in the browser and you're ready to play the most complicated forms of TicTacToe.

For this game the frontend is build using Angular with the style beeing done with Tailwind CSS.

For the backend I used NestJS with the socket.io library to transfere the current game state live from one player to the other.

The app supports multiple games around the globe.

A 2d-platformer mobile game I designed, developed and deployed during and after my abitur.

You play the little ball and you can move and jump from all floors, walls and ceillings. But watch out not to fall down or die from the obstacles in the course.

It's possible to play this game on time, where you need to master each stage in the given order or to play in an endless mode, where all stages appear in random order one after another.

Get the lowest time or the highest score to set new highscores boths gamemodes.

The game is built using the Unity Unity Engine and C#

Sadly the game is not available in the AppStore and PlayStore so it's not possible to play the game as easy.
